Tag

The Tag component is used to categorize items. The text inside the Tag should be kept as short as possible. Multi-labels should be avoided as well.

Properties

NameTypeDefault
dismissible

boolean

Determins whether the dismiss functionality is enabled
onDismiss

() => void

The function to handle when the dismiss icon is clicked		-

The Tag supports margin styled-system props.

<Tag>Filter</Tag>
