Textarea

The Textarea supports margin, height and width styled-system props.

Properties

NameTypeDefault
variant

'boxed' | 'bottom-lined'

The variant of the component.
size

'small' | 'medium'

The size of the component.
inverted

boolean

Determines whether the component follows the inverted color scheme.
label

string

Label for the input.
placeholder

string

Placeholder for the component.
error

boolean

Determines the error state of the component.
